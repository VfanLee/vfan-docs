# 正则表达式（JavaScript）

## 1. 创建正则表达式

### 1.1. 字面量创建正则表达式

```js
const reg = /foo/g
```

### 1.2. 对象创建正则表达式

```js
const reg = new RegExp('foo', 'g')
```
