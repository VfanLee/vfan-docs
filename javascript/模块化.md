# 模块化

## 1. IIFE

```js
// 定义模块
(function () {
  var userInfo = {
    id: 1,
    username: "vfanlee",
    password: "vfanlee",
  };

  function sayHi(name) {
    console.log("hello " + name);
  }

  window.module1 = {
    userInfo,
    sayHi,
  };
})();

// 访问模块
console.log(window.module1)
```

## 2. AMD

RequireJS 是 AMD 规范的实现方案。以下是 RequireJS 的基本使用：

### 2.1. 准备工作

<!-- tabs:start -->
<!-- tab:index.html -->
引入 RequireJS，并定义 RequireJS 配置文件。

```html
<script data-main="./main.js" src="https://cdn.jsdelivr.net/npm/requirejs@2.3.6/require.min.js"></script>
```

<!-- tab:main.js -->
RequireJS 配置文件。

```js
requirejs.config({
  baseUrl: ".",
  paths: {},
});
```
<!-- tabs:end -->

### 2.2. 定义模块

语法：`define(name, deps, callback)`

```js
// 定义 MainController 模块
define([], function () {
  function MainController() {}

  return MainController;
});

```

### 2.3. 引入模块

语法：`require(deps, callback, errback, optional)`

```js
// 引入 mainController.js 模块
require(['./MainController.js'], function (MainController) {
  console.log(MainController);
});
```

## 3. ES6

### 3.1. 导出模块

```js
export default xxx

export const aaa = function () {}
export const bbb = function () {}
```

### 3.2. 导入模块

```js
// 加载内置依赖
import MODULE from 'MODULE'

import xxx from 'PATH/xxx'

import { aaa, bbb } from 'PATH/MODULE'
import { aaa as A, bbb as B } from 'PATH/MODULE'
import * as AB from 'PATH/MODULE'
```
