# Vue I18n

> 参考文档：https://github.com/intlify/vue-i18n-next#readme  
> 当前版本：v9

## 1. 安装依赖

**注意：vue3 请使用 v9 版本。**

```bash
npm i vue-i18n@9
```

## 2. locale 配置

参考如下目录结构创建对应文件。

```markdown
- locale      # i18n 配置文件夹
  - index.js  # i18n 配置文件
  - zh.json   # 中文包
  - en.json   # 英文包
  - ...       # 其他语言包
- main.js     # 项目入口
```

### 2.1. i18n 配置文件

```js
import { createI18n } from 'vue-i18n'
import zh from './zh.json'
import en from './en.json'
import Cookies from 'js-cookie'

const getLocale = () => {
  return Cookies.get('locale') || 'zh'
}

const messages = {
  zh,
  en,
}

const i18n = createI18n({
  locale: getLocale(),  // set locale
  messages,             // set locale messages
  legacy: false,        // you must set `false`, to use Composition API
})

export default i18n
```

### 2.3. 引入 i18n 配置文件

```js
import { createApp } from 'vue'

import i18n from './locale'

createApp(App)
  .use(i18n)
  .mount('#app')
```

## 3. 使用示例

```vue
<script setup>
import Cookies from 'js-cookie'
import i18n from '@/locale'

const locales = [
  {
    id: 'zh',
    name: '简体中文',
  },
  {
    id: 'en',
    name: 'English',
  },
]

const chooseLocale = locale => {
  Cookies.set('locale', locale)
  // 更改 Global Scope locale
  i18n.global.locale.value = locale
}
</script>

<template>
  <main>
    <div>{{ $t('test1', { aaa: 'hhh' }) }}</div>
    <div>{{ $t('test2') }}</div>
    <div>{{ $t('test3') }}</div>

    <div class="locale-changer">
      <!-- 
        v-model 绑定 Local Scope locale
        Local Scope 默认继承于 Global Scope
      -->
      <select
        v-model="$i18n.locale"
        @change="chooseLocale($event.target.value)"
      >
        <option v-for="locale in locales" :key="locale.id" :value="locale.id">
          {{ locale.name }}
        </option>
      </select>
    </div>
  </main>
</template>
```
