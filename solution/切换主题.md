# 主题切换

## 切换 css 文件

1. 使用多个独立的 css 文件来定义 css 变量。

    <!-- tabs:start -->
    <!-- tab:定义 light 变量 -->
    ```css
    :root {
      --body-bg-color: #fff;
      --box-bg-color: #000;
    }
    ```

    <!-- tab:定义 dark 变量 -->
    ```css
    :root {
      --body-bg-color: #000;
      --box-bg-color: #fff;
    }
    ```
    <!-- tabs:end -->

2. 通过切换 css 变量文件来切换主题。

    <!-- tabs:start -->
    <!-- tab:HTML -->
    ```html
    <!DOCTYPE html>
    <html lang="en">
      <head>
        <meta charset="UTF-8" />
        <meta name="viewport" content="width=device-width, initial-scale=1.0" />
        <title>Document</title>
        <link rel="stylesheet" href="./index.css" />
        <!-- 初始化（默认 light 主题） -->
        <link id="theme" rel="stylesheet" href="./theme/light.css" />
      </head>
      <body>
        <button id="theme__btn--light">切换明亮主题</button>
        <button id="theme__btn--dark">切换暗黑主题</button>
        <div class="box"></div>

        <script class="./index.js"></script>
      </body>
    </html>
    ```

    <!-- tab:CSS -->
    ```css
    body {
      background-color: var(--body-bg-color);
    }

    .box {
      width: 100px;
      height: 100px;
      background-color: var(--box-bg-color);
    }
    ```

    <!-- tab:JS -->
    ```js
    // 切换 light 主题
    document.getElementById('theme__btn--light').addEventListener('click', () => {
      document.querySelector('link#theme').setAttribute('href', './theme/light.css')
    })

    // 切换 dark 主题
    document.getElementById('theme__btn--dark').addEventListener('click', () => {
      document.querySelector('link#theme').setAttribute('href', './theme/dark.css')
    })
    ```
    <!-- tabs:end -->

## 切换 css Class 类名

1. 使用 css 主题文件，采用类名进行区分主题。

    ```css
    :root.light {
      --body-bg-color: #fff;
      --box-bg-color: #000;
    }

    :root.dark {
      --body-bg-color: #000;
      --box-bg-color: #fff;
    }
    ```

2. 通过切换 Class 类名来切换主题。

    <!-- tabs:start -->
    <!-- tab:HTML -->
    ```html
    <!DOCTYPE html>
    <!-- 初始化（默认 light 主题） -->
    <html lang="en" class="light">
      <head>
        <meta charset="UTF-8" />
        <meta name="viewport" content="width=device-width, initial-scale=1.0" />
        <title>Document</title>
        <link rel="stylesheet" href="./theme.css" />
      </head>
      <body>
        <button id="theme__btn--light">切换明亮主题</button>
        <button id="theme__btn--dark">切换暗黑主题</button>
        <div class="box"></div>
      </body>
    </html>
    ```

    <!-- tab:CSS -->
    ```css
    body {
      background-color: var(--body-bg-color);
    }

    .box {
      width: 100px;
      height: 100px;
      background-color: var(--box-bg-color);
    }
    ```

    <!-- tab:JS -->
    ```js
    // 切换 light 主题
    document.getElementById('theme__btn--light').addEventListener('click', () => {
      document.documentElement.className = 'light'
    })

    // 切换 dark 主题
    document.getElementById('theme__btn--dark').addEventListener('click', () => {
      document.documentElement.className = 'dark'
    })
    ```
    <!-- tabs:end -->
