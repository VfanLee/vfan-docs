# 获取文件

## `<input type="file" />` 上传文件

`<input type="file" />` 元素可通过 `event.target.files` 获取上传文件列表。

```html
<input type="file" id="file-input-element" />
```

### 方式1：使用传统的 DOM 选择器访问一个被选择的文件

```js
const fileEl = document.getElementById('file-input-element')
const files = fileEl.files
// 获取文件后执行后续操作
```

### 方式2：通过 change 事件访问被选择的文件

```js
const fileEl = document.getElementById('file-input-element')
fileEl.addEventListener("change", e => {
  const files = e.target.files
  // 获取文件后执行后续操作
})
```

## `<input type="file" />` 样式优化

在实际使用过程中，由于上传样式需要自定义，所以通常会隐藏 `<input type="file" />`，需要 **间接使用** 隐藏的 `<input type="file" />` 元素。

### 方式1：通过 `click()` 方法使用隐藏的 `<input type="file" />` 元素

input 元素添加 `display: none` 的样式，再手动调用 input 元素的 `click()` 方法完成上传操作。

```html
<input id="file-input-element" type="file" style="display:none">
<!-- #file-select 以按钮为例，可以是任何样式 -->
<button id="custom-upload">选择文件</button>
```

```js
const fileEl = document.getElementById("file-input-element")
const customUpload = document.getElementById("custom-upload")

customUpload.addEventListener("click", e => {
  fileEl && fileEl.click()
})

fileEl.addEventListener("change", e => {
  // 执行后续操作
})
```

### 方式2：使用 label 元素来触发一个隐藏的 `<input type="file" />` 元素

考虑到键盘可访问性，可以通过点击 `label` 元素来完成上传操作。

> **注意**：input 元素不能使用 `display: node` 或者 `visibility: hidden` 来隐藏，否则 label 将无法通过键盘访问。

```html
<input type="file" id="file-input-element" class="visually-hidden">
<!-- label 可以是任何样式 -->
<label class="custom-upload" for="file-input-element">选择文件</label>
```

```js
const fileEl = document.getElementById("file-input-element")

fileEl.addEventListener("change", e => {
  // 执行后续操作
})
```

## `Drag & Drop` 上传文件

可能会存在拖拽文件到指定区域进行上传操作，那么就可以借助 `Drag & Drop` 事件来进行上传操作。

可通过 `drop` 事件 `event.dataTransfer.files` 获取上传文件列表。

```html
<!-- 初始化拖放区域 -->
<div id="dropbox-element"></div>
```

```js
let dropEl = document.getElementById("dropbox-element")

dropEl.addEventListener("dragenter", e => {
  e.stopPropagation()
  e.preventDefault()
})

dropEl.addEventListener("dragover", e => {
  e.stopPropagation()
  e.preventDefault()
})

dropEl.addEventListener("drop", e => {
  e.stopPropagation()
  e.preventDefault()

  const files = e.dataTransfer.files
  // 获取文件后执行后续操作
})
```
