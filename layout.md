# 管理系统布局方案

## 左右布局

### 支持特性

- 左侧支持收缩/展开。
- 右侧宽高自适应。

### 实现分析

- 整体采用 `flex` 布局，布局方向为 `row`。
- 左侧定宽则通过 vue3 的 `v-bind` 实现响应式属性和 css 变量的绑定，并设置 `flex-shrink: 0;` 防止缩小。
- 右侧自适应通过 `flex: 1;` 实现，由于布局方向是 row，所以需要将 `width: 0;`，确保 `flex: 1;` 的有效性。

## 上下布局

### 支持特性

- 上侧定高。
- 下侧宽高自适应。

### 实现逻辑

- 整体采用 flex 布局，布局方向为 `column`。
- 上部定高则通过 vue3 的 `v-bind` 实现响应式属性和 css 变量的绑定，并设置 `flex-shrink: 0;` 防止缩小。
- 中部自适应通过 `flex: 1;` 实现，由于布局方向是 column，所以需要将 `height: 0;`，确保 `flex: 1;` 的有效性。
