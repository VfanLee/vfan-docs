# Vite 创建项目

vite 可以选择多种项目，如：vanilla js、vue、react ……

```sh
npm create vite

# 然后根据需求选择不同项目
```

## 创建 Vue 项目

可以使用官方脚手架 [create-vue](https://github.com/vuejs/create-vue) 来进行创建。

使用如下命令来创建 Vue3 项目

```sh
npm create vue@3

# or【目前最新为 vue3】
npm create vue@latest
```

使用如下命令来创建 Vue2 项目

```sh
npm create vue@2
```
