# Vue3

## 基本使用

```js
import { createApp } from 'vue'

import App from './App.vue'
import router from './router'
import { createPinia } from 'pinia'

const app = createApp(App)

app.use(router)
app.use(createPinia())

app.mount('#app')
```

## 单文件组件 CSS 功能

> 参考 [单文件组件 CSS 功能](https://cn.vuejs.org/api/sfc-css-features.html)

### 组件作用域 CSS

使用 `:deep` 进行深度选择。

```css
.a :deep(.b) {
  /* ... */
}

/* 上面的代码会被编译成 */
.a[data-v-f3f3eg9] .b {
  /* ... */
}
```

```css
:deep {
  .b {
    /* ... */
  }
}
```

### CSS 中的 `v-bind()`

```html
<script setup>
const theme = {
  color: 'red'
}
</script>

<template>
  <p>hello</p>
</template>

<style scoped>
p {
  color: v-bind('theme.color');
}
</style>
```
