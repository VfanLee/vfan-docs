# Vue I18n

> 参考文档：https://github.com/intlify/vue-i18n-next#readme  
> 当前版本：v9

## 1. 安装依赖

**注意：vue3 请使用 v9 版本。**

```sh
npm i vue-i18n@9
```

## 2. locale 配置

参考如下目录结构创建对应文件。

```markdown
- locale      # i18n 文件夹
  - index.js  # i18n 配置文件
  - zh.json   # 中文包
  - en.json   # 英文包
- main.js     # 项目入口
```

### 2.1. i18n 配置文件

```js
import { createI18n } from 'vue-i18n'
import zh from './zh.json'
import en from './en.json'

const messages = {
  zh,
  en,
}

const i18n = createI18n({
  locale: 'zh',         // set locale
  messages,             // set locale messages
  legacy: false,        // you must set `false`, to use Composition API
})

export default i18n
```

### 2.3. 引入 i18n 配置文件

```js
import { createApp } from 'vue'

import i18n from './locale'

createApp(App)
  .use(i18n)
  .mount('#app')
```

## 3. 使用示例

### 3.1. template 中使用国际化字段

```vue
<template>
  <!-- 当前语言 -->
  <div>{{ $i18n.locale }}</div>

  <!-- 使用国际化语言字段 -->
  <div>{{ $t('test1') }}</div>
  <div>{{ $t('test2') }}</div>
  <div>{{ $t('test3') }}</div>
</template>
```

### 3.2. js 中使用国际化字段

```js
import i18n from '@/locale'

const { t } = i18n.global

// 使用国际化语言字段
console.log(t('test1'))
console.log(t('test2'))
console.log(t('test3'))
```

### 3.3. 更改语言

```js
import i18n from '@/locale'

const { locale } = i18n.global

// 更改为英文包
locale.value = 'en'
```
