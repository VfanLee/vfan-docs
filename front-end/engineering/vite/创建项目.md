# Vite 创建项目

## 创建 Vue 项目

可以使用官方脚手架 [create-vue](https://github.com/vuejs/create-vue) 来进行创建。

### 创建 Vue3 项目

```sh
npm create vue@latest

# or 可能会存在有 npm 缓存而不是最新的
npm create vue@3
```

### 创建 Vue2 项目

```sh
npm create vue@2
```

## 创建其他项目

vite 可以快速创建多种项目，如：vanilla JS、vue、react ……

```sh
npm create vite
```
