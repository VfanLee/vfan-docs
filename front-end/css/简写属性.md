# CSS 简写属性

> 参见：[CSS 的简写属性](https://developer.mozilla.org/zh-CN/docs/Web/CSS/Shorthand_properties)

## all

CSS 属性 [all](https://developer.mozilla.org/en-US/docs/Web/CSS/all) 将除了 `unicode-bidi` 与 `direction` 之外的所有属性重设至其初始值，或继承值。

- `initial` 关键字用于设置 CSS 属性为它的默认值，也就是浏览器的初始样式。例如：`color: initial` 就相当于 `color: black`，因为 black 是 color 属性的默认值。
- `inherit` 关键字用于设置 CSS 属性为它的父元素的值，也就是继承父元素的样式。例如：`color: inherit` 就相当于把 color 属性设置为和父元素一样的颜色。
- `unset` 关键字用于设置 CSS 属性为不固定的值，也就是根据属性的继承特性来决定。
  - 如果该属性是默认继承属性，那么 unset 等同于 inherit；例如：`color: unset` 就相当于 `color: inherit`，因为 color 是可继承属性。
  - 如果该属性是非继承属性，那么 unset 等同于 initial。例如：`border: unset` 就相当于 `border: initial`，因为 border 是不可继承属性。
- `revert` 关键字用于设置 CSS 属性为它的原始值，也就是恢复到之前的样式。
  - 如果该属性有用户代理（浏览器）的样式，那么 revert 等同于用户代理的样式。例如：`color: revert` 就相当于恢复到浏览器默认的颜色。
  - 如果该属性没有用户代理的样式，那么 revert 等同于 initial。例如：`border: revert` 就相当于恢复到没有边框的样式。

## list-style

- `list-style-image`: 列表样式图像。
- `list-style-position`: 列表样式位置。
- `list-style-type`: 列表样式类型。

## border

`border-width`: 定义元素的边框厚度。
`border-style`: 定义元素的边框样式。
`border-color`: 定义元素的边框颜色。

## flex

- `flex-grow`：用来指定扩展比率，即剩余空间是正值时此「flex子项」相对于「flex容器」里其他「flex子项」能分配到空间比例。
- `flex-shrink`：用来指定收缩比率，即剩余空间是负值时此「flex子项」相对于「flex容器」里其他「flex子项」能收缩的空间比例。
- `flex-basis`：用来指定伸缩基准值，即在根据伸缩比率计算出剩余空间的分布之前，「flex子项」长度的起始数值。

- 如果缩写 `flex: 1`, 则其计算值为 `1 1 0%`。
- 如果缩写 `flex: auto`, 则其计算值为 `1 1 auto`。
