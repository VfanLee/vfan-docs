## 清除浮动

.clearfix::after {
  content: '';
  display: block;
  clear: both;
  /* 兼容性 */
  height: 0;
  visibility: hidden;
}

/* ::before 作用是为了解决外边距塌陷问题 */
.clearfix::before,
.clearfix::after {
  content: '';
  display: table;
}
.clearfix::after {
  clear: both;
  /* 兼容性 */
  height: 0;
  visibility: hidden;
}
