# Node.js 安装

Node.js 官方地址：https://nodejs.org/zh-cn

## Windows

### nvm

> 项目地址：https://github.com/coreybutler/nvm-windows

```sh
nvm ls # 列出已安装的 node.js
nvm ls available # 列出可用版本
nvm install <version_id> # 安装指定版本node
nvm use <version_id> # 使用指定版本node
nvm uninstall <version_id> # 卸载指定版本node
```
