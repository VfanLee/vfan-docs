# 基本使用

## 1. 创建一个 Vue 应用

<!-- tabs:start -->
<!-- tab:vue -->
```js
import { createApp } from 'vue'

import App from './App.vue'
import router from './router'
import { createPinia } from 'pinia'

const app = createApp(App)

app.use(router)
app.use(createPinia())

app.mount('#app')
```

<!-- tab:vue2 -->
```js
import Vue from 'vue'

import App from './App.vue'
import router from './router'
import store from './store'

const app = new Vue({
  router,
  store,
  render: h => h(App),
}).$mount('#app')
```
