# 验机步骤

验机时最好戴上手套避免沾染指纹

## 1. 包装

机器一般有两个包装：外包装、内包装。

### 1.1. 外包装

1. 检查包装是否破损。（因运输原因，小面积破损并无大碍）
2. 检查型号是否一致。
3. 检查封条是否受损。

### 1.2. 内包装

1. 检查包装是否破损。
2. 检查型号是否一致。

## 2. 开箱

1. 检查是否有三包凭证。
2. 检查机器是否沾染指纹。（主要看电源键、触摸板、接口附近）
3. 检查接口是否有插拔痕迹。
4. 检查螺丝是否有拧动痕迹。

## 3. 开机

并不是每个机器都有运输模式，所以开机时可以插电源。

切记不要联网，因为联网就代表激活。

若开机时没有跳过联网选项，如何跳过联网步骤？

1. `Shift + F10` 或者 `Shift + Fn + F10` 唤出命令行。
2. 命令行输入：`oobe\bypassnro`，完成后机器会重启，此时就会有跳过联网的步骤了。

## 4. 图吧工具箱验机

> 下载链接：http://www.tbtool.cn/

1. 查看机器型号，是否一致。
2. `硬盘工具` 中 `DiskInfo` 工具查看硬盘通电时间（50小时以内）和通电次数（2位数以内）。
3. `CPU工具` 中 `AIDA64` 查看电源损耗程度（损耗程度个位数还算正常）。
4. `屏幕工具` 中 `坏点漏光测试` 检查屏幕有无坏点（3个以内算正常）。

无误后，基本是安全下车了！
