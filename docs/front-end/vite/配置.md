### 配置 alias 别名

> 参考：[配置 resolve.alias](https://cn.vitejs.dev/config/shared-options.html#resolve-alias)

<!-- tabs:start -->
<!-- tab:对象写法 -->
```js
resolve: {
  alias: {
    '@': fileURLToPath(new URL('./src', import.meta.url))
  }
}
```

<!-- tab:数组写法 -->
```js
resolve: {
  alias: [
    {
      find: '@',
      replacement: fileURLToPath(new URL('./src', import.meta.url))
    }
  ]
}
```
<!-- tabs:end -->
