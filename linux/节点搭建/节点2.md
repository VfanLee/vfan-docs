4. 域名解析。

## 部署 x-ui

### ssl 证书申请

```bash
x-ui
16 # 一键申请 ssl 证书，根据提示一步一步来即可
```

成功之后，在 `/root/cert` 下可以查看申请的证书。

![](assets/2023-03-16-21-53-40.png)

接着在 x-ui 面板中填写公钥、私钥路径即可。
