# nginx 实例

## server 配置多个 location

注意：

- 当 server 配置多个 location 时，其他 location 不能使用 root 而需要使用 alias。
- root 不仅限制于 location 中，也可以写在 server 中。
- alias 只限制于 location 中。

```conf
server {
	listen 80;
	server_name test.example.com;

	location / {
		root	/var/www/test;
		index	index.html;
	}

	location /foo {
		alias	/var/www/foo;
		index	index.html;
	}

	location /bar {
		alias	/var/www/bar;
		index	index.html;
	}
}
```

## 配置代理

`proxy_ssl_server_name`：当开启时，在 SSL 握手的时候会把 HostName 传给上游服务器，以便上游服务器知道用哪个证书。
`proxy_pass`：代理地址。

```conf
server {
	listen	443 ssl;
	server_name	test.example.com;
	ssl_certificate	/root/ssl_example.com/test/cert.crt;
	ssl_certificate_key	/root/ssl_example.com/test/private.key;
	root	/var/www/app/test/dist/;

	location / {
		try_files	$uri $uri/ /index.html;
		index	index.html;
	}

	location /g {
		proxy_ssl_server_name	on;
		proxy_pass	https://google.com;
	}
}
```

## 配置 ssl 证书

`ssl_certificate`：SSL 证书公钥。
`ssl_certificate_key`：SSL 证书私钥。

```conf
server {
	listen	443 ssl;
	server_name	test.example.com;
	ssl_certificate	/root/ssl_example.com/test/cert.crt;
	ssl_certificate_key	/root/ssl_example.com/test/private.key;

	location / {
		root	/var/www/test;
		index	index.html;
	}
}
```

## 重定向

> `rewrite` 类似于 `return`，`return` 执行效率更高。

```conf
# 将 http://example.com、http://www.example.com 重定向至 https://example.com
server {
	listen	80;
	server_name	example.com www.example.com;
	return	301 https://example.com;
}

# 将 https://www.example.com 重定向至 https://example.com
server {
	listen	443 ssl;
	server_name	www.example.com;
	ssl_certificate	/root/ssl_example.com/www/cert.crt;
	ssl_certificate_key	/root/ssl_example.com/www/private.key;
	return	301 https://example.com;
}

server {
	listen	443 ssl;
	server_name	example.com;
	ssl_certificate	/root/ssl_example.com/cert.crt;
	ssl_certificate_key	/root/ssl_example.com/private.key;

	location / {
		root	/var/www/app/resume/dist;
		try_files	$uri $uri/ /index.html;
		index	index.html;
	}
}
```

## 配置用户可访问的静态服务器

```conf
server {
	listen	80;
	server_name	test.example.com;

	location / {
		alias	/opt/www/;
		autoindex	on; // 可访问目录
	}
}
```
