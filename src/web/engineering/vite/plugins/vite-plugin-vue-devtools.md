# vite-plugin-vue-devtools

## 介绍

提升 vue 开发体验

## 安装

```sh
npm install -D vite-plugin-vue-devtools
```

## 使用

```js
import { defineConfig } from 'vite'
import vueDevTools from 'vite-plugin-vue-devtools'

export default defineConfig({
  plugins: [vueDevTools()],
})
```
