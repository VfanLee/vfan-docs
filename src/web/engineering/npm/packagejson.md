# package.json

## 语义版本控制 (SemVer)

- Patch 补丁版本：`1.0` 或 `1.0.x` 或 `~1.0.4`
- Minor 次要版本：`1` 或 `1.x` 或 `^1.0.4`
- Major 主要版本：`*` 或 `x`

::: warning
- **`^`**：不会升级到主版本（例如 `^1.0.0` 会更新到 `1.x.x`，但不会更新到 `2.0.0`）。
- **`~`**：不会升级到次版本（例如 `~1.0.4` 会更新到 `1.0.x`，但不会更新到 `1.1.0`）。
- **`*` 或 `x`**：允许安装任何版本。
:::
