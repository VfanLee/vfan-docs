# 语法综述

## 词类

1. verb（动词）
2. noun（名词）
3. article（冠词）
4. Pronoun（代词）
5. adjective（形容词）
6. numeral（数词）
7. adverb（副词）
8. preposition（介词）
9. interjection（叹词）
10. conjunction（连词）

### 动词分类

谓语动词

非谓语动词

助动词

## 句子成分

1. 主语
2. 谓语动词
3. 宾语
4. 宾语补语
5. 主语补语
6. 定语
7. 状语
8. 同位语

## 基本句型

1. 主语 + 不及物动词
2. 主语 + 单及物动词 + 宾语
3. 主语 + 双及物动词 + 间接宾语 + 直接宾语
4. 主语 + 复杂及物动词 + 宾语 + (宾语)补语
5. 主语 + 系动词 + (主语)补语/表语
